Exercise 4 - Fruit vs Fruit Monster 2.0

In this sketch I converted the creature functions into a custom MGMonster class with interactive controls for Fruit vs Fruit Monster 2.0.

Controls:
• Click monsters to change color (uses fruit color palette)
• ↑/↓ arrow keys: increase/decrease size for all monsters
• R key: reset all monsters
• M key: create new monster at mouse position
• B key: create new balloon at mouse position

This exercise demonstrates object-oriented programming with classes, arrays, vector movement, and interactive controls.

FRUIT vs FRUIT MONSTER 2.0 QNMonster class includes parent and child display methods, transform controls, and multiple action methods for user interaction.
